# ヴェルダラ：砕かれた門

## 実行環境の必要条件
* Python >= 3.10
* pygame >= 2.1
* 外部画像・外部音声ファイルは使用していないため，追加の素材ダウンロードは不要

## ゲームの概要
* おにぎりと戦います
*　銃を使って戦います
*　敵を倒したら経験値がもらえます
*　経験値が一定数たまるとレベルアップします
*　レベルアップした場合、ステータスのどれかを一つ上昇させるか、スキルがもらえます。

## ゲームの遊び方
* `W`，`A`，`S`，`D`キーまたは矢印キーでプレイヤーを移動する
* `K`キーで近接攻撃を行う
*　`J`で銃を撃ちます
* 経験値が一定量に達するとレベルアップし，HP，MP（弾の数），Attack，Speed　スキルのどれかを一つ選び得ることができる
*　`R`でリロードしてMPを満タンにします
*　スキルは一度消費すると再使用まで一定時間必要　
* スキル画面、ステータス画面は右上に表示されています
*　目標は左上に表示されています
* 目標を達成した場合にボスが出現します（予定）
* ボスは大きくデザインされています。
*　銃の種類、弾の数は右下に表示されています
 
* 通常画面で`ESC`キーを押すとゲームを終了する
* HPが0になるとゲームオーバーになる

## ゲームの実装
### 共通基本機能
* Pygameによるゲームウィンドウとゲームループ
* 見下ろし型マップの描画
* プレイヤーの移動と画面内での位置制限
* 敵の追跡，攻撃，当たり判定
* HP，MP，EXP，レベルアップの管理
* 基本GUI（HPバー，MPバー，EXPバー，ミニマップ，クエスト表示，ホットバー）
* `os.chdir(os.path.dirname(os.path.abspath(__file__)))` により，実行場所に依存しないファイルパス処理

### 分担追加機能
* スキルの追加
* 銃の種類を増やす
* mob敵の種類を増やす
* ステージを追加する
* ボスの追加2体(担当：C0A25120 島田 寛生)：ステージ１のボスとステージ２のボスを作成
### メモ
* ボスの出現の仕方は仮で敵を10体倒すと出現に設定されてある。
* ボスを倒したときのステージ2への移動は未設計
